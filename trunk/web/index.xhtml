DataPET 3.0

Conheça a nova versão do DataPET!

Bem vindo ao DataPET 3.0 RC1! Nesta versão contamos com um visual inteiramente novo e tecnologias de interface com usuário mais modernas para facilitar a interação do usuário com o sistema. O software foi inteiramente refeito seguindo a arquitetura MVC e conta com várias novas funcionalidades, como o sistema de anotações e os gráficos de notas.

Nesta nova versão, o DataPET deixa de ser uma simples ferramenta para a geração de relatórios para se tornar um sistema de gerenciamento de grupos PET. O sistema se propõe a ser o braço direito do tutor na avaliação das atividades de seu grupo e dos petianos que a realizam. É com esta nova vertente que o DataPET segue desta versão em diante, e com isso esperamos a utilização do sistema por toda a Universidade Federal de Uberlândia e posteriormente para as demais universidades do Brasil.
